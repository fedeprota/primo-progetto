# Pre-Launch Conversion Checklist

Verify your landing page before launch.

---

## Section Completeness

### All 14 Sections Present

- [ ] **Section 1: Hero** - Clear value proposition visible in 3 seconds
- [ ] **Section 2: Trust Strip** - Specific numbers or recognizable logos
- [ ] **Section 3: Emotional Reframe** - "It's not your fault" messaging
- [ ] **Section 4: Problem Mirror** - 5-7 specific symptoms listed
- [ ] **Section 5: Consequences** - Cost of inaction clearly stated
- [ ] **Section 6: False Solutions** - 3+ alternatives disqualified
- [ ] **Section 7: Core Differentiator** - Unique insight/approach explained
- [ ] **Section 8: Outcomes** - Specific, measurable results listed
- [ ] **Section 9: Implementation Areas** - Scope/coverage clear
- [ ] **Section 10: Process** - 3-step methodology explained
- [ ] **Section 11: Social Proof** - Testimonials with names and logos
- [ ] **Section 12: ROI/Urgency** - Cost-per-day of inaction quantified
- [ ] **Section 13: FAQ** - Top 5-7 real objections addressed
- [ ] **Section 14: Final CTA** - Single clear action to take

### Section Order

- [ ] Sections appear in correct psychological sequence (1-14)
- [ ] No sections skipped or combined inappropriately

---

## Copy Quality

### Tone

- [ ] High-conviction consulting tone (not salesy or weak)
- [ ] "You" appears more than "we"
- [ ] Active voice used throughout
- [ ] No hedging language ("might", "could", "potentially")

### Specificity

- [ ] All metrics are specific numbers (not "many" or "some")
- [ ] Testimonials include specific results
- [ ] Outcomes are quantified where possible
- [ ] Trust strip has concrete metrics

### Problem-Agitate-Solve

- [ ] Problem is named specifically (not generic)
- [ ] Consequences are amplified
- [ ] Solution is positioned as logical answer

### False Alternative Disqualification

- [ ] At least 3 alternatives addressed
- [ ] Each alternative has specific reason for failure
- [ ] Your approach positioned as only remaining option

---

## CTA Verification

### Placement

- [ ] CTA in Hero section (above fold)
- [ ] CTA after Outcomes section (mid-page)
- [ ] CTA in Final section (bottom)
- [ ] Optional: Sticky CTA on scroll

### Copy

- [ ] Action-oriented verb ("Book", "Get", "Start")
- [ ] Risk reducer included ("Free", "No obligation")
- [ ] Clear next step stated
- [ ] Button text matches destination

### Functionality

- [ ] All CTA buttons work
- [ ] Links go to correct destination
- [ ] Calendar/form loads properly
- [ ] Mobile CTA tap targets are large enough

---

## Social Proof Quality

### Testimonials

- [ ] Include specific before/after results
- [ ] Author name and title included
- [ ] Company name or logo visible
- [ ] Photo included (if available)

### Logos

- [ ] Logos are recognizable to target audience
- [ ] Logo quality is high (not pixelated)
- [ ] Permission obtained for logo use

### Metrics

- [ ] Trust strip metrics are accurate
- [ ] Numbers are impressive but believable
- [ ] Metrics are relevant to target audience

---

## Objection Handling (FAQ)

### Coverage

- [ ] Price/value objection addressed
- [ ] Time investment objection addressed
- [ ] Fit objection addressed ("Is this right for me?")
- [ ] Risk objection addressed ("What if it doesn't work?")
- [ ] Timing objection addressed ("Should I wait?")

### Quality

- [ ] Answers are direct and confident
- [ ] No hedging or weak responses
- [ ] Answers reinforce value proposition

---

## Technical & UX

### Mobile Experience

- [ ] All sections readable on mobile
- [ ] Images scale properly
- [ ] CTA buttons are tappable
- [ ] Text is large enough to read
- [ ] Psychological flow maintained on mobile

### Load Performance

- [ ] Page loads in < 3 seconds
- [ ] Images are optimized
- [ ] No broken assets

### Accessibility

- [ ] Sufficient color contrast
- [ ] Alt text on images
- [ ] Headings properly structured

---

## Final Review

### The 3-Second Test

Show someone the page for 3 seconds, then ask:
- [ ] Can they tell WHO this is for?
- [ ] Can they tell WHAT problem you solve?
- [ ] Can they tell WHAT to do next?

### The Scroll Test

Scroll through the page quickly:
- [ ] Is the visual hierarchy clear?
- [ ] Do headlines tell the story alone?
- [ ] Is the CTA obvious at each scroll position?

### The Objection Test

Ask a skeptical friend to review:
- [ ] What objections do they still have?
- [ ] Are those objections addressed in the FAQ?

---

## Pre-Launch Actions

- [ ] Test all forms/calendars with real submission
- [ ] Check analytics/tracking is installed
- [ ] Set up thank-you/confirmation page
- [ ] Prepare follow-up email sequence
- [ ] Brief team on how to handle inquiries

---

## Post-Launch Monitoring

### First 24 Hours

- [ ] Check for broken links/forms
- [ ] Monitor form submissions
- [ ] Watch for console errors

### First Week

- [ ] Review scroll depth analytics
- [ ] Check CTA click rates
- [ ] Note any user feedback

### Ongoing

- [ ] A/B test headlines
- [ ] Refine based on objections heard in calls
- [ ] Update testimonials as new ones come in
